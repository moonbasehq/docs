# Moonbase HTTP API

Moonbase exposes an HTTP API that can be used to headlessly interact with Moonbase data plane.

Moonbase roughly follows [JSON API](https://jsonapi.org/) standards.

## OpenAPI

You can view the current OpenAPI documentation [here](https://api.moonbasehq.com/v1/openapi.json).

## API Limits

```
Max payload size : 5242880 (5Mib)
Rate limit       : 200k requests per 20 minutes.
```
