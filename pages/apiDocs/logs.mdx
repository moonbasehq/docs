## /logs

### GET

- Operation ID: logs-list
- Summary: Get team logs
- Description: Gets logs for a given team

#### Parameters

| Name | In | Required | Type | Description |
| ---- | -- | -------- | ---- | ----------- |
| q | query | false | undefined | A search query to find logs by. |
| projectIds | query | false | undefined | Comma delimited list of projectIds. Ex 'cuid1,cuid2,cuid3'. |
| start | query | false | string |  |
| end | query | false | string |  |
| cursor | query | false | undefined | Indicates the next offset to start searching from. |
| teamId | query | false | string |  |
| limit | query | false | undefined |  |
| rel | query | false | string |  |

#### Responses

**200**:
- Description: Successful response
