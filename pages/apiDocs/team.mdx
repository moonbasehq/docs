## /:teamId/projects/:id

### GET

- Operation ID: projects-getById
- Summary: Create project
- Description: Gets a project by a unique ID.

#### Parameters

| Name | In | Required | Type | Description |
| ---- | -- | -------- | ---- | ----------- |
| teamId | path | true | string | ID of the team that the project belongs to. |
| id | path | true | string | ID of the project being fetched. |

#### Responses

**200**:
- Description: Successful response
##### Content

**application/json**:
```json
{

  "properties": {
    "id": {
      "type": "string",
      "pattern": "^[cC][^\\s-]{8,}$"
    },
    "name": {
      "type": "string",
      "minLength": 1,
      "maxLength": 120,
      "nullable": true
    },
    "teamId": {
      "type": "string",
      "pattern": "^[cC][^\\s-]{8,}$"
    },
    "slug": {
      "type": "string",
      "minLength": 1
    },
    "description": {
      "type": "string",
      "nullable": true
    },
    "environment": {
      "type": "string",
      "nullable": true
    },
    "createdAt": {
      "type": "string",
      "format": "date-time"
    }
  },
  "required": [
    "id",
    "name",
    "teamId",
    "slug",
    "description",
    "environment",
    "createdAt"
  ],
  "additionalProperties": false
}
```
